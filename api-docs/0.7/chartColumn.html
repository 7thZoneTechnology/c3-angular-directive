directive: chartColumn

chartColumn

chart-column Used to provide data values for the chart as well as the name and some other config options.

Restrict To: Element

Parent Element: c3chart

Attributes:
Name Type Description
column-id String

The id used to uniquely identify the column
column-values String

The values for this column to plot.
column-type String

The type of the column to show: line, spline, bar, step, area, area-spline, area-step, scatter, pie, donut, gauge

c3js docs
column-name String

The name of the column as used to print in the label.

c3js docs
column-color String

The color to use for this column.

c3js docs
Example
Usage:
  <chart-column column-values="..."/>

Example:
  http://jettro.github.io/c3-angular-directive/#examples

<chart-column column-id="data 1"
              column-name="Data 1"
              column-color="red"
              column-values="30,200,100,400,150,250"
              column-type="spline"/>
