directive: chartAxisXTick

chartAxisXTick

chart-axis-x-tick is used to customize the x axis tick properties. You can change the amount of ticks, the format of the tick, culling, rotating.

Restrict To: Element

Parent Element: chart-axis-x

Attributes:
Name Type Description
tick-count Number

Specify the number of ticks on the x axis.

c3js doc
tick-culling Boolean

Culling means not all ticks will be shown, for category data this is by default false, for other data true.

c3js doc
tick-culling-max Number

Set the maximum number of ticks, if specified culling is by default true.

c3js doc
tick-multiline Boolean

Not sure what this does, not documented.

c3js doc
tick-centered Boolean

Centers the tick on the x axis

c3js doc
tick-rotate Number

Number of degrees to rotate the tick, can also be a negative number.

c3js doc
tick-fit Boolean

Default is to make the tick fit the chart, if false it will be at the exact position of the x value.

c3js doc
tick-outer Boolean

Default is not to show the outer tick, setting this to true will show the outer tick.

c3js doc
tick-values Array

An array containing the exact values to present a tick for.

c3js doc
tick-format function

Provide a d3 based format for the tick value. format: '$,'

c3js doc
tick-format-function function

Provide a function to format the tick value. format: function (d) { return '$' + d; }

c3js doc
