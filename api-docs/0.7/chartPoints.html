directive: chartPoints

chartPoints

chart-points is used configure the points in for example a line chart. You can configure the radius of the point in normal as well as expand state.

Restrict To: Element

Parent Element: c3chart

Attributes:
Name Type Description
showPoint Boolean

Whether to show points in the chart.

c3js docs
pointExpandEnabled Boolean

Whether to expand on mouse over.

c3js docs
pointExpandRadius Number

Radius of the point when expanded. Default is 1.75 times the normal radius.

c3js docs
pointRadius Number

Radius of the point in normal mode. Default radius is 2.5

c3js docs
pointSelectRadius Number

Radius of the point when selected, default is 4 times the normal radius.

c3js docs
Example
Usage:
  <chart-points show-point="..." point-expand-enabled="..."/>
