directive: chartTooltip

chartTooltip

chart-tooltip is used to configure the look and feel of the tooltip. You can configure to show the tooltip or not and the formatting of labels, values, etc.

Restrict To: Element

Parent Element: c3chart

Attributes:
Name Type Description
showTooltip Boolean

Whether to show the tooltip or not.

c3js docs
hideTooltipTitle Boolean

Whether to show the tooltip title or not.
groupTooltip Boolean

Whether to group all tooltips of the different columns in the chart.

c3js docs
titleFormatFunction function

Function to format the title of the tooltip.

c3js docs
nameFormatFunction function

Function to format the name of the tooltip.

c3js docs
valueFormatFunction function

Function to format the value of the tooltip.

c3js docs
