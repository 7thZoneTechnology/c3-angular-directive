directive: chartGrid

chartGrid

chart-grid is used to specify properties to show a grid.

Restrict To: Element

Parent Element: c3chart

Attributes:
Name Type Description
showX Boolean

Whether to show the x axis grid.

c3js docs
showY Boolean

Whether to show the y axis grid.

c3js docs
showY2 Boolean

Whether to show the y2 axis grid.

c3js docs
showFocus Boolean

Whether to enable the focus grid.

c3js docs
Example
Usage:
  <chart-grid showX="..." showY="..." showY2="..."/>

Example:
  http://jettro.github.io/c3-angular-directive/#examples

<chart-grid show-x="false" show-y="true">
  <chart-grid-optional axis-id="x" grid-value="1" grid-text="Start"/>
  <chart-grid-optional axis-id="y" grid-value="20" grid-text="Minimum"/>
  <chart-grid-optional axis-id="y" grid-value="200" grid-text="Maximum"/>
</chart-grid>
