directive: chartAxes

chartAxes

chart-axes is used to customize the axes properties. Using this directive you can select the propertie(s) to use for the different categories or for the time field. You can also configure for the different columns to use y or y2.

Restrict To: Element

Parent Element: c3chart

Attributes:
Name Type Description
valuesX String

Specify the key in the data object to use for the x value

http://c3js.org/reference.html#data-x
valuesXs String

Specify the different keys for different data columns in format columnId:key,columnId:key

http://c3js.org/reference.html#data-xs
y String

Set the id(s) of columns to use the first y value (y). Format is comma separated.

http://c3js.org/reference.html#data-axes
y2 String

Set the id(s) of columns to use the second y value (y2) Format is comma separated.

http://c3js.org/reference.html#data-axes
Example
Usage:
  <chart-axes values-x="..." values-Xs="..." y="..." y2="..."/>
