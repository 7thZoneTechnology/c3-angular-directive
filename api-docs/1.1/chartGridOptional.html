directive: chartGridOptional

chartGridOptional

chart-grid-optional is used to add optional grid lines to the chart. All attributes are mandatory

Restrict To: Element

Parent Element: gridDirective

Attributes:
Name Type Description
axisId String

x, y or y2.

c3js docs
value Number

Value where to print the additional grid line.

c3js docs
text String

Label to print at the grid line.

c3js docs
gridClass String

Class to add to the grid line to be able to style them separately.

c3js docs
position String

Sets the position for the label, values are: start, middle, end.

c3js docs
Example
Usage:
  <chart-grid-optional axis-id="..." value="..." text="..."/>

Example:
  http://jettro.github.io/c3-angular-directive/#examples

<chart-grid show-x="false" show-y="true">
  <chart-grid-optional axis-id="x" grid-value="1" grid-text="Start"/>
  <chart-grid-optional axis-id="y" grid-value="20" grid-text="Minimum"/>
  <chart-grid-optional axis-id="y" grid-value="200" grid-text="Maximum"/>
</chart-grid>
