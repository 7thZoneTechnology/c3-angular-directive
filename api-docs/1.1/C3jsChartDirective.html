C3jsChartDirective

C3jsChartDirective()

AngularJS Directive for the c3js library. With this directive we make it easier to work with the great c3js libary. We provide a number of directives that make it possible to declaritively create some charts.

This documentation can be found online at: http://jettro.github.io/c3-angular-directive/api-docs/0.7/C3jsChartDirective.html

More information about the directive can be found here http://jettro.github.io/c3-angular-directive/
