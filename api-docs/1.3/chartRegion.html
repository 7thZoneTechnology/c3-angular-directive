directive: chartRegion

chartRegion

chart-region is used to set a region property on a chart.

Restrict To: Element

Parent Element: c3chart

Attributes:
Name Type Description
region-id String

The id used to uniquely identify the column
region-style String

Style to identify the regions.

c3js doc
region-starts String

The regions where the data starts.
region-ends String

The regions where the data starts.
Example
Usage:
  <chart-region region-id="..." region-style="..." region-starts="..." region-ends="..."/>
