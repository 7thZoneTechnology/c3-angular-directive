directive: chartEvents

chartEvents

chart-events Used to provide callback functions to respond to events of the charts.

Restrict To: Element

Parent Element: c3chart

Attributes:
Name Type Description
on-init function

The on init callback function.

c3js docs
on-rendered function

Provide the callback to respond to on-rendered. Basically, this callback will be called in each time when the chart is redrawed.

c3js docs
on-mouseover function

Provide callback to be called when you hoover the chart.

c3js docs
on-mouseout function

Provide callback to be called when you hoover out of the chart.

c3js docs
on-resize function

Provide callback to be called when the chart is resizing.

c3js docs
on-resized function

Provide callback to be called when the chart is resized.

c3js docs
on-click-data function

Provide callback to be called one of the data points, lines, bars, etc. is clicked.

c3js docs
on-mouseover-data function

Provide callback to be called one of the data points, lines, bars, etc. is hoovered.

c3js docs
on-mouseout-data function

Provide callback to be called one of the data points, lines, bars, etc. is hoovered out.

c3js docs
